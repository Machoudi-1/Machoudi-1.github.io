# Portfolio Source

Development repository of my personal portfolio.

This repository contains the complete source code, development assets and build pipeline used to generate the production website.

The public website is available at:

**🌐 https://machoudi-1.github.io**

---

## Overview

This portfolio showcases my work as an AI R&D Engineer with a focus on:

- Artificial Intelligence
- Machine Learning
- Computer Vision
- MLOps
- Data Science

The project is developed as a maintainable software application rather than a simple static website.

---

## Repository Structure

```text
portfolio-source/
├── assets/                 # Images, icons and static resources
├── index.html              # Development HTML
├── style.css               # Development stylesheet
├── script.js               # Application logic
├── config.js               # External links and configuration
├── package.json
├── package-lock.json
├── README.md
└── LICENSE
```

The production website is generated separately and published to the public repository.

---

## Development

Install dependencies:

```bash
npm install
```

Build the production assets:

```bash
npm run build
```

## Objectives

This repository is designed to:
